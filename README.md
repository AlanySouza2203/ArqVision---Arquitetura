# 🏛️ ArqVision – Site de Arquitetura

## 📌 Sobre o Projeto

O **ArqVision** é um site institucional desenvolvido para apresentar um estúdio de arquitetura fictício.  
O objetivo do projeto é demonstrar projetos arquitetônicos, a filosofia do escritório e criar uma experiência visual moderna para visitantes interessados em arquitetura e design.

O site foi desenvolvido utilizando **HTML, CSS e JavaScript**, com foco em layout moderno, imagens em destaque e navegação simples.

---

# 🚀 Tecnologias Utilizadas

- **HTML5** – Estrutura do site  
- **CSS3** – Estilização, layout e animações  
- **JavaScript** – Interatividade do slider de imagens  
- **Font Awesome** – Ícones de redes sociais

## 🖼️ Página de Projetos

A página de projetos apresenta uma **galeria interativa de arquitetura**.

- Slider de imagens em tela cheia  
- Miniaturas clicáveis

- Destaque visual no projeto selecionado
